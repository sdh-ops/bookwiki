# 북위키 (Book-Wiki) 사이트 안내서

## 개요
북위키는 출판업계 종사자를 위한 커뮤니티 플랫폼입니다.
구인구직, 지원사업 정보를 자동으로 수집하고, 회원 간 정보 공유가 가능합니다.

---

## 주요 기능

### 1. 게시판
| 게시판 | 설명 |
|--------|------|
| **구인구직** | 북에디터(bookeditor.org) 구인글 자동 수집 |
| **지원사업** | 출판진흥원(kpipa.or.kr) 지원사업 자동 수집 + 마감 캘린더 |
| **자유게시판** | 회원 자유 게시글 |
| **AI허브** | AI 기반 출판 도구 (개발 중) - 원서검토, 번역, 마케팅 등 |
| **HOT** | 주간 인기 게시물 (조회수 + 댓글수 기반) |

### 2. 자동 스크래핑
- **업데이트 주기**: 2시간마다 자동 실행 (GitHub Actions)
- **중복 방지**: 원본 URL 기준으로 중복 없이 업데이트
- **데이터 범위**: 2026년 3월 1일 이후 게시물

### 3. 지원사업 마감 캘린더
- 지원사업 게시판 상단 "목록/캘린더" 탭으로 전환
- 마감일이 있는 지원사업을 달력 형태로 표시
- 다가오는 마감 목록 자동 표시
- 마감일 추출: 원문의 "기간: YYYY-MM-DD ~ YYYY-MM-DD" 패턴에서 종료일 추출

### 4. 회원 기능
- **로그인**: 이메일 매직링크 방식 (비밀번호 없음)
- **글쓰기**: 로그인 없이도 비밀번호로 작성 가능
- **수정/삭제**: 본인 글 또는 비밀번호 입력으로 관리

### 5. 주간 베스트
- 최근 7일간 인기 게시물 자동 선정
- HOT 점수 = 조회수 + (댓글수 × 10)

---

## 관리자 기능
- `/admin` 페이지에서 전체 게시물 관리
- 자동 수집 게시물(is_auto) 일괄 관리 가능
- 관리자 계정은 bw_admins 테이블에서 관리

---

## 기술 스택
- **프론트엔드**: Next.js 14 (App Router)
- **데이터베이스**: Supabase (PostgreSQL)
- **배포**: Vercel
- **자동화**: GitHub Actions

---

## 데이터베이스 구조

### bw_posts (게시물)
| 컬럼 | 설명 |
|------|------|
| id | 고유 ID |
| title | 제목 |
| content | 본문 (HTML) |
| author | 작성자 |
| board_type | 게시판 유형 (job/support/free/ai) |
| source_url | 원본 URL (자동수집 게시물) |
| is_auto | 자동수집 여부 |
| view_count | 조회수 |
| comment_count | 댓글수 |
| created_at | 작성일 |

### bw_comments (댓글)
| 컬럼 | 설명 |
|------|------|
| id | 고유 ID |
| post_id | 게시물 ID |
| content | 댓글 내용 |
| author | 작성자 |
| created_at | 작성일 |

## URL 구조
| 경로 | 설명 |
|------|------|
| `/` | 메인 (전체 게시물) |
| `/?board=job` | 구인구직 |
| `/?board=support` | 지원사업 |
| `/?board=free` | 자유게시판 |
| `/?board=ai` | AI허브 |
| `/?board=hot` | HOT 게시물 |
| `/post/[id]` | 게시물 상세 |
| `/write` | 글쓰기 |
| `/login` | 로그인 |
| `/admin` | 관리자 페이지 |
